# ClinicaWorkspace

Monorepo para el sistema clínico del examen IND-01.

## Estructura del Proyecto

- **api-clinica/**: Aplicación Backend desarrollada con NestJS.
- **web-clinica/**: Aplicación Frontend desarrollada con Next.js (CSS y Playwright).
- **prisma/**: Configuración del ORM Prisma y esquema de base de datos PostgreSQL.
- **.env**: Configuración de variables de entorno locales (Base de Datos).
- **.gitignore**: Exclusiones de archivos del repositorio.
